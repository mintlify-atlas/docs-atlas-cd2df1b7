## What is Reverse API Engineer?

Reverse API Engineer is a CLI tool that eliminates the tedium of manual API reverse engineering. Instead of inspecting network requests by hand and writing boilerplate code from scratch, you simply browse a website while the tool records all network traffic — then AI analyzes the captured data and generates a clean, production-ready API client automatically.

No more digging through browser DevTools. No more manually copying headers and request bodies. Just browse, capture, and get code.

## How it works

<Steps>
  <Step title="Browse and capture">
    Open a browser session with HAR recording enabled. Navigate to the website and interact with it — log in, search, click, scroll — to trigger the API calls you want to capture.
  </Step>
  <Step title="Close the browser">
    When you're done, close the browser. All network traffic has been saved as a HAR (HTTP Archive) file.
  </Step>
  <Step title="AI generates your client">
    Claude analyzes the HAR file, identifies API endpoints, authentication patterns, and request/response schemas, then generates clean Python (or JavaScript/TypeScript) code.
  </Step>
  <Step title="Use your API client">
    Your generated scripts are saved to `./scripts/{task_name}/` with an API client, usage examples, and README documentation.
  </Step>
</Steps>

## Operating modes

| Mode | Description | Best for |
|------|-------------|----------|
| **agent** | Autonomous AI-driven browser navigation | Automatable tasks where you describe the goal |
| **manual** | You browse, AI generates the client | Complex sites needing human navigation |
| **engineer** | Re-process an existing HAR capture | Iterating on previous sessions |
| **collector** | AI-powered web data collection to JSON/CSV | Structured data extraction |

Switch between modes with **Shift+Tab** in the interactive CLI.

## Requirements

- Python 3.11+
- An `ANTHROPIC_API_KEY` (for Claude SDK) or a running OpenCode instance
- Playwright browsers (installed separately after pip install)

<Note>
  This tool executes AI-generated code locally using Claude Code. Always review generated scripts before running them in production environments.
</Note>
